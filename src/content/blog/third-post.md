Astro is a comfortable fit for personal publishing because it handles the boring parts well. Markdown is first-class, content collections provide structure, and the default output stays extremely fast.

## What I reach for first

I usually start with the official blog template, then trim it back:

- simplify the navigation
- reduce decorative elements
- tighten the archive layout
- add only the features the writing actually needs

## What comes later

Comments, analytics, and external integrations are useful, but they are not required for the first useful version of a blog.
