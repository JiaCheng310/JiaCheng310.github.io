A personal blog does not need much to be useful. It needs a stable URL, readable typography, and a publishing workflow that does not feel heavier than the note itself.

## Why minimal still works

The best writing tools often disappear while you are using them. If the page loads quickly and the layout stays calm, the reader spends more time with the paragraph and less time negotiating the interface.

## What I want from the stack

- Markdown for drafting
- A simple content collection for structure
- Static output for speed and portability
- Search and tags for retrieval later

## The tradeoff

Minimal design is not the same thing as no design. It still needs hierarchy, spacing, and enough personality to feel intentional.
